# MYCYBERBASE

This is a static website project (HTML/CSS/JS) for the MYCYBERBASE site.

How to push to GitHub

1. Create a repository on GitHub (name it e.g. `MYCYBERBASE`).
2. In this folder run the commands shown after the initial commit:

```bash
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```

Replace `USERNAME/REPO` with your repository path.

Notes
- Update `git config user.name` and `git config user.email` to your info if needed.
- If you prefer using `gh` (GitHub CLI) you can create the repo with `gh repo create`.
